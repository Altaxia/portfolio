IUT pixel war

✕

Temps de réalisation

3 semaines

Date

Avril 2024

Nombre de personnes impliquées

1

Dans le cadre d'un projet de réalisation avec JavaScript, dans le second semestre de la première année d'IUT, nous avons tous du réaliser une interface en HTML/CSS sans frameworks pour pouvoir accéder à une reproduction de la "Pixel War" de reddit. Pour cela, nous avions tous accès à une API documentée, que nous devions intégrer à notre code en utilisant JavaScript.
